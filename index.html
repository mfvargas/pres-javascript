JavaScript

    JavaScript (JS) es un lenguaje de programación utilizado para añadir interactividad a las páginas web.

    Junto con HTML y CSS, JS es una de las tres principales tecnologías para la producción de contenido en la Web.

Historia de JavaScript

    Fue creado en 1995 por Brendan Eich, con el fin de dar interactividad a las páginas web leídas con el navegador Netscape.

    En la actualidad, es utilizado por todos los principales navegadores web y en la mayoría de los sitios web.

Características de JavaScript

  • Se ejecuta en el cliente (i.e. navegador web)
  • Es un lenguage interpretado
  • Permite el uso de varios paradigmas de programación, incluyendo la programación orientada a objetos

¿Qué puede hacerse con JavaScript?

  • Controlar multimedia
  • Reaccionar a "eventos" (ej. presionar un botón)
  • Modificar dinámicamente el contenido de una página web
  • ¡Manejar mapas interactivos!
  • Y muchas, muchas, cosas más...

"Hola mundo" en JavaScript


                            <!DOCTYPE html>
                            <html lang="es">
                                <head>
                                    <title>Hola mundo</title>
                                    <meta charset="UTF-8"> 
                                    <script>
                                        function holaMundo () {
                                            alert("¡Hola mundo!");
                                        }        
                                    </script>
                                </head>
                                <body>
                                    <button type="button" onclick="holaMundo()">
                                        Presióneme
                                    </button>        
                                </body>
                            </html>

El tag <script>

    El tag <script> se utiliza para incluir un script en JS. Puede contener el código o una referencia a un archivo.

    
                                <script>
                                    function holaMundo () {
                                        alert("¡Hola mundo!");
                                    }        
                                </script>
                                
                                <script src="js/misfunciones.js"></script>
                                
                                <script src="https://maps.googleapis.com/maps/api/js"></script>

Eventos en HTML y JS

    Los eventos son fragmentos de código que se activan con la ocurrencia de sucesos como, por ejemplo: terminar la carga de una página web, llenar un campo de entrada de datos o presionar un botón. 

    
                                <button type="button" onclick="alert('¡Hola mundo!');">
                                    Presióneme
                                </button>

El Document Object Model

El Document Object Model

    El Document Object Model (DOM) es una representación jerárquica de los objetos que componen una página web.

    Es también una interfaz de programación que permite que un programador tenga acceso para leer y modificar los elementos, atributos, estilos y eventos de un documento HTML.

El atributo id

    El atributo id establece un identificador único para cada elemento HTML de una página. Es utilizado en JS para acceder programáticamente al DOM.

    
                                <button type="button" 
                                onclick="document.getElementById('fechaYhora').innerHTML = Date();">
                                    Presióneme
                                </button>
                                
                                <p id="fechaYhora"></p>

El atributo class

    El atributo class establece la pertenencia de un elemento HTML a un grupo (o clase).

    
                                <script>
                                  function changeColor() {
                                    var x = document.getElementsByClassName("clase01");
                                    var i;
                                    for (i = 0; i < x.length; i++) {
                                      x[i].style.backgroundColor = "red";
                                    }
                                  }
                                </script>

                                <button type="button" onclick="changeColor();">Presióneme</button>		
                                
                                <h1 id="id01" class="clase01">Primer encabezado</h1>
                                <h1 id="id02" class="clase01">Segundo encabezado</h1>

Programación Orientada a Objetos (POO)

    Los objetos son entidades que poseen:

    • Estado: representado por un conjunto de variables llamadas propiedades.
    • Comportamiento: representado por un conjunto de funciones llamadas métodos.

    Los objetos pertenecen a grupos llamados clases.

POO: clases

    Todos los objetos de una clase comparten los mismos métodos y propiedades.

    Una clase puede verse como una plantilla o “machote” a partir de la cual se crean objetos.

    A un objeto creado a partir de una clase se le llama también una instancia de esa clases.

POO: constructores

    En JS, un nuevo objeto (o instancia) de una clase se con la palabra reservada new. 

    
                                var person1 = new Person('Bob');
                                var person2 = new Person('Sarah');

    En este ejemplo, se crean dos objetos de la clase "Person" y se inicializan con los nombres de las personas.

JS: listas de propiedades

    En JS, los objetos (y otras estructuras) pueden representarse mediante listas de tuplas de la forma propiedad:valor 

    
                                var person = {firstName:"John", lastName:"Doe", age:30, eyeColor:"green"};

    En este ejemplo, se especifican cuatro propiedades de una persona, con sus respectivos valores.
