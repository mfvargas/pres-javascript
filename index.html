JavaScript

    JavaScript (JS) es un lenguaje de programación utilizado para añadir interactividad a las páginas web.

    Junto con HTML y CSS, JS es una de las tres principales tecnologías para la producción de contenido en la Web.

Historia de JavaScript

    Fue creado en 1995 por Brendan Eich, con el fin de dar interactividad a las páginas web leídas con el navegador Netscape.

    En la actualidad, es utilizado por todos los principales navegadores web y en la mayoría de los sitios web.

Características de JavaScript

  • Se ejecuta en el cliente (i.e. navegador web)
  • Es un lenguage interpretado
  • Permite el uso de varios paradigmas de programación, incluyendo la programación orientada a objetos

¿Qué puede hacerse con JavaScript?

  • Controlar multimedia
  • Reaccionar a "eventos" (ej. presionar un botón)
  • Modificar dinámicamente el contenido de una página web
  • ¡Manejar mapas interactivos!
  • Y muchas, muchas, cosas más...

El tag <script>

    El tag <script> se utiliza para incluir un script en JS. Puede contener el código o una referencia a un archivo.

    
                                <script>
                                    /* Función que imprime la hilera "¡Hola mundo!" */
                                    function holaMundo () {
                                        alert("¡Hola mundo!");
                                    }        
                                </script>
                                
                                <script src="js/misfunciones.js"></script>
                                
                                <script src="https://maps.googleapis.com/maps/api/js"></script>

Declaración e inicialización de variables

    Se declaran con la palabra reservada var

    Se inicializan con el operador =

    
                                var lenguaje;
                                var numeroAureo, pi;
                                var esBisiesto = false;                                
                                
                                lenguaje = "JavaScript";
                                numeroAureo = 1.61803, pi = 3.14;

    El operador = se usa también para actualizar los valores de las variables, después de inicializarlas

Tipos de datos: númericos

    Incluye números tanto enteros como decimales.

Tipos de datos: textuales

    Son hileras de texto escritas entre comillas simples o dobles.

Tipos de datos: booleanos

    Tienen el valor de verdadero (true) o falso (false).


                            var esBisiesto = true;
                            var esLunes = false;
                            
                            // "Y" lógico (AND)
                            2 > 1 && 2 == (1 + 1); // Resultado: true;

                            // "O" lógico (OR)
                            1 > 2 || 4 < 3; // Resultado: false;
                            
                            // "No" lógico (NOT)
                            !(2 > 1); // Resultado: false;

Tipos de datos: arreglos (arrays)

    Son objetos con múltiples valores separados por comas.


                            var nombres = ['Carlos', 'Lucía', 'Jorge'];
                            var numeros = [20, 9, 10.3];
                            
                            nombres[0]; // Resultado: 'Carlos'
                            numeros[2]; // Resultado: 10.3

Objetos

    Son estructuras compuestas por listas de tuplas de la forma propiedad:valor

Funciones

    Declaración

    
                      function [nombre-función] ([arg1], [arg2], ..., [argn]) {       
                        [cuerpo-función]
                        
                        return [valor-retorno]
                      }

    Ejemplo

    
                      function cuadrado (numero) {
                        c = numero * numero;
                        
                        return c;
                      }
                      
                      cantidad = cuadrado(8);

Condicionales: if

    Sintaxis

    
                      if ([condición]) {
                        [código a ejecutarse si la condición es verdadera];       
                      }

    Ejemplo

    
                      var x = 2, y = 4;
                      if (x < y) {
                        alert("x es menor o igual que y");
                      }

Condicionales: if - else

    Sintaxis

    
                      if ([condición]) {
                        [código a ejecutarse si la condición es verdadera];       
                      } else {
                        [código a ejecutarse si la condición es falsa];       
                      }

    Ejemplo

    
                      var x = 2, y = 4;
                      if (x < y) {
                        alert("x es menor o igual que y");
                      } else {
                        alert("x es mayor o igual que y");
                      }

if - else if - else

    
                      if ([condición-1]) {
                        [código a ejecutarse si la condición-1 es verdadera];
                      } else if [condición-n]{
                        [código a ejecutarse si la condición-n es verdadera];
                      } else {
                        [código a ejecutarse si todas las condiciones son falsas];        
                      }

    Ejemplo

    
                      var x = 2, y = 4;
                      if (x < y) {
                        alert("x es menor o igual que y");
                      } else if (x > y) {
                        alert("x es mayor que y");
                      } else {
                        alert("x es igual a y");
                      }

Condicionales: switch

    Sintaxis

    
                    switch ([expresión]) {
                      case [opción-1]:
                        [código a ejecutarse para la opción-1]
                        break;
                      case [opción-n]:
                        [código a ejecutarse para la opción-n]
                        break;
                      default:
                        [código a ejecutarse si no se cumple ninguna opción]
                    }

    break detiene la ejecución e impide que las otras opciones sean evaluadas

Condicionales: switch

    Ejemplo

    
                    codigoMes = 1;
                    switch (codigoMes) {
                      case 1:
                        nombreMes = "Enero";
                        break;
                      case 2:
                        nombreMes = "Febrero";
                        break;
                      case 3:
                        nombreMes = "Marzo";
                        break;                        
                      default:
                        nombreMes = "Otro";
                    }
                    alert(nombreMes);

Ciclos: for

    Sintaxis

    
                    for ([expresión-inicial];[condición]; [expresión-incremento]) {          
                      [cuerpo-ciclo]
                    }

    Ejemplo

    
                    // Imprime los cuadrados de i, para i en [1..3]
                    for (i = 1; i <= 3; i++) {          
                      alert(i * i);
                    }

Ciclos: while

    Sintaxis

    
                    while ([condición]) {          
                      [cuerpo-ciclo]
                    }

    Ejemplo

    
                    // Imprime los cuadrados de i, para i en [1..3]           
                    var i = 1;
                    while (i <= 3) {          
                      alert(i * i);
                    }

Ciclos: for..in

    Sintaxis

    
                    for ([variable] in [objeto]) {          
                      [cuerpo-ciclo]
                    }

    Ejemplo

    
                    // Imprime los valores de las propiedades de un objeto
                    var perro = {nombre:'Lassie', raza:'Collie', sexo:'Femenino', edad:7 };           
                    for (i in perro) {          
                      alert(perro[i]);
                      i++;
                    }

Ciclos: do..while

    Sintaxis

    
                    do {          
                      [cuerpo-ciclo]
                    } while ([condición]);

    Ejemplo

    
                    // Imprime los cuadrados de i, para i en [1..3]           
                    var i = 1;
                    do {          
                      alert(i * i);
                      i++;
                    } while (i <= 3);

Eventos en HTML y JS

    Los eventos son fragmentos de código que se activan con la ocurrencia de sucesos como, por ejemplo: terminar la carga de una página web, llenar un campo de entrada de datos o presionar un botón. 

    
                                <button type="button" onclick="alert('¡Hola mundo!');">
                                    Hola mundo
                                </button>

El Document Object Model

El Document Object Model

    El Document Object Model (DOM) es una representación jerárquica de los objetos que componen una página web.

    Es también una interfaz de programación que permite que un programador tenga acceso para leer y modificar los elementos, atributos, estilos y eventos de un documento HTML.

Acceso al DOM mediante JS

  1. Observe el ejemplo en CodePen
  2. Cree una cuenta gratuita en CodePen
  3. Bifurque (fork) el ejemplo hacia su cuenta
  4. Reproduzca el ejemplo en tres archivos en su computadora: index.html, css/estilos.css y js/funciones.js. Recuerde establecer (con <link> y <script>) los enlaces a los otros archivos desde index.html.

El atributo id

    El atributo id establece un identificador único para cada elemento HTML de una página. Es utilizado en JS para acceder programáticamente al DOM.

    
                                <button type="button" 
                                onclick="document.getElementById('fechaYhora').innerHTML = Date();">
                                    Presióneme
                                </button>
                                
                                <p id="fechaYhora"></p>

El atributo class

    El atributo class establece la pertenencia de un elemento HTML a un grupo (o clase).

    
                                <script>
                                  function changeColor() {
                                    var x = document.getElementsByClassName("clase01");
                                    var i;
                                    for (i = 0; i < x.length; i++) {
                                      x[i].style.backgroundColor = "red";
                                    }
                                  }
                                </script>

                                <button type="button" onclick="changeColor();">Presióneme</button>		
                                
                                <h1 id="id01" class="clase01">Primer encabezado</h1>
                                <h1 id="id02" class="clase01">Segundo encabezado</h1>

Programación Orientada a Objetos (POO)

    Los objetos son entidades que poseen:

    • Estado: representado por un conjunto de variables llamadas propiedades.
    • Comportamiento: representado por un conjunto de funciones llamadas métodos.

    Los objetos pertenecen a grupos llamados clases.

POO: clases

    Todos los objetos de una clase comparten los mismos métodos y propiedades.

    Una clase puede verse como una plantilla o “machote” a partir de la cual se crean objetos.

    A un objeto creado a partir de una clase se le llama también una instancia de esa clases.

POO: constructores

    En JS, un nuevo objeto (o instancia) de una clase se con la palabra reservada new. 

    
                                var person1 = new Person('Bob');
                                var person2 = new Person('Sarah');

    En este ejemplo, se crean dos objetos de la clase "Person" y se inicializan con los nombres de las personas.

Recomendaciones para programar en JS

  • Tabule con dos espacios
  • Finalice todas las sentencias con punto y coma (;)
  • Divida la lógica del programa en funciones y preferiblemente también en clases y objetos
  • Mantenga el código JS en un archivo separado de la sintaxis de HTML y CSS
